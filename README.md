# Antiquity Alive

НА ЭТАПЕ РАЗРАБОТКИ, МНОГИЕ ФУНКЦИИ НЕ РЕАЛИЗОВАНЫ ИЛИ РЕАЛИЗОВАНЫ С ЦЕЛЬЮ ТЕСТИРОВАНИЯ И ОБУЧЕНИЯ. 

## Описание проекта

Проект "Antiquity Alive" - это информационно-справочный сайт, посвященный истории античности (Древняя Греция, Древний Рим, эллинистические государства). Сайт предоставляет пользователям удобный доступ к проверенной информации, включая статьи, хронологию, биографии исторических личностей и описания артефактов.

## Функционал

- Навигация по разделам: Главная, Хронология, Персоналии, Статьи, Добавление данных.
- Полнотекстовый поиск по статьям и персоналиям.
- Динамическое отображение контента без перезагрузки страницы (SPA).
- Адаптивный дизайн для различных устройств.
- Возможность добавления новых статей и персоналий (в демо-режиме с alert).

## Технологический стек

- **Фронтенд:** React 18
- **Стили:** CSS
- **Сборка:** Vite
- **Хостинг:** Vercel / Netlify (статические файлы)

## Установка и запуск
# Клонировать репозиторий
git clone https://github.com/weool/antiquity-alive.git

# Перейти в папку проекта
cd antiquity-alive

# Установить зависимости
npm install

# Запустить проект
npm run dev
